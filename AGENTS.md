# AGENTS

この project は mozyo-bridge scaffold rules を使用する。

## セッション開始

1. 現在の working directory がこの project root またはその配下であることを確認する。
2. mozyo-bridge の central preset rules を読む:
   - `${MOZYO_BRIDGE_HOME:-~/.mozyo_bridge}/rules/presets/asana/agent-workflow.md`
3. project-local docs (例: target project の README、codebase 規約 docs、運用 docs) が存在する場合は読む。これらの path は target project 側の任意の convention であり、mozyo-bridge の runtime 必須参照ではない。
4. 非自明な作業を始める前に active な Asana task を確認する。

## Preset

- ticket system: `asana`
- preset version: `2026.05.12.1`
- generated by: `mozyo-bridge 0.1.9`

## Guardrails

- `${MOZYO_BRIDGE_HOME:-~/.mozyo_bridge}/rules/presets/asana/agent-workflow.md` を取得できない場合は、読んだふりをせず停止する。
- Asana task state と task comment を durable な作業状態として扱う。pane message と chat message は通知にすぎない。
- credential、token、個人情報を repository file や task comment に記録しない。
